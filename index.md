Currently, he's working at [Walmart Labs](https://www.walmartlabs.com/){:target="_blank"}. He's interested in the following topics:

* Distributed Systems

* Algorithm Design and PL
* Designing REST APIs
* Building Scalable Systems
* Machine Learning and NLP
